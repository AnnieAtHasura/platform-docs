Pro-tier clusters
=================
..
   - What is a prod cluster?
   - Outline - creating and modifying a cluster
   - 
   - Provisioning

A pro-tier cluster is a :doc:`Hasura cluster <../index>` running on the Hasura
pro-tier i.e. dedicated infrastructure managed by Hasura. The configuration of
the cluster is declarative, and hence can be version controlled in a Hasura
project.

.. note::

  * For pricing details for the pro-tier, see `Hasura pricing <https://platform.hasura.io/pricing>`_.
